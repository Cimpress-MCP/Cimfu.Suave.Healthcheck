A set of functions for operating with the HealthSwitch type.

Function or valueDescription

Disables the switch so that it will report Unhealthy during a healthcheck.

Signature: hs:HealthSwitch -> unit

Returns the healthcheck associated with a switch.

Signature: disabledMessage:string option -> HealthSwitch

Creates a switch which, when disabled, will report the provided disabledMessage.
serverMain
Signature: HealthSwitch

Main default switch for the whole server. Enabling or disabling this swich will cause Checks.serverMainSwitch to report Healthy or Unhealthy respectively.
